概述

产品/组合列表

接口描述（ ）

接口名称 列表
接口地址 /Goods/lists
请求方式 POST
数据格式 JSON

请求参数

<公共传入参数>

编码 名称 类型 说明 默认值
goods_type 类型 string 产品类型(产品：goods,组合：package) 请求套餐时需要传递，默认为产品

返回参数

<公共返回参数>

编码 名称 类型 说明 默认值
code 状态码 int 返回状态码，用于调试 无
status 状态 bool 返回状态：true-成功，false-失败 无
data 返回数据 object 暂无 无

参数项：data

编码 名称 类型 说明 默认值
id 产品自增ID int 产品自增ID 无
name 产品名称 string 产品名称 无
title 产品简介 string 产品简介 无
picture 产品封面图 string 图片链接 无
picture_more 产品介绍图 array 详情图片 无
sales 产品累计销量 int 总销量 无
sales_month 月销量 int 月销量 无
browse 浏览量 int 浏览量 无
stock 库存 int 库存 无
price 实际价格 string 实际价格 无
mark_price 市场价格 string 市场价格 无
video_id 视频编号 string 视频编号 无
